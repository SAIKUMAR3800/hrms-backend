# HRMS Backend

This project is a lightweight **Human Resource Management System (HRMS Lite) backend** built using **FastAPI** and **PostgreSQL**.  
It provides RESTful APIs to manage employees and track attendance.

The backend is designed to work with the Angular frontend and supports CRUD operations for employees and attendance management.

---

# Project Overview

The HRMS Lite backend allows an admin to:

- Manage employee records
- Track employee attendance
- Store data in a PostgreSQL database
- Provide RESTful APIs for the frontend application

---

# Tech Stack

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Uvicorn

# Setup Instructions

## 1. Navigate to Backend Folder

```bash
cd hrms-backend
2. Create Virtual Environment
python -m venv venv
3. Activate Virtual Environment
Windows
venv\Scripts\activate
Mac / Linux
source venv/bin/activate
4. Install Dependencies
pip install -r requirements.txt
Running the Backend Server

To start the FastAPI development server run:

python -m uvicorn main:app --reload

Once the server is running, open your browser and navigate to:

http://127.0.0.1:8000
